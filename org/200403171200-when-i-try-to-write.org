#+filetags: :poetry:
#+title: When I try to write

#+BEGIN_VERSE
When I try to write
I consider millions of minds,
and how each would judge the words I choose.
How hard it is for a man to think alone;
how impossible to guess the opinions of a multitude.
#+END_VERSE
